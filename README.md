# GSK x DigData Step Up Career Challenge

**Live demo:** https://mulualem03.github.io/gsk-step-up-challenge/MiraculonRx_prototype.html

End-to-end analysis and prototype for a Phase-3 oncology trial of a novel drug (Miraculon-B), completed as part of the GSK Step Up virtual work experience programme.

## Headline findings
- Miraculon-B nearly doubled tumour-response rate vs standard of care (55.2% vs 32.6%, p < 10^-9).
- Baseline protein concentration is the dominant predictor of both response and adverse events.
- Logistic regression model AUC = 0.86 under 5-fold cross-validation.

## Deliverables
- `TaskA_Miraculon-B_Analysis.pptx`: clinical trial analysis presentation (Path A).
- `TaskB_MiraculonRx_Design.pptx`: design thinking presentation for the web tool (Path B).
- `MiraculonRx_prototype.html`: working single-file prototype of the prescriber decision-support web app. Open in any browser.
- `Project_Retrospective.md`: reflective retrospective using Gibbs' cycle.

## Tech
Python (pandas, scikit-learn, matplotlib), HTML/CSS/JavaScript, PowerPoint.
